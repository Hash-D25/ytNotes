YouTube Notes Dashboard

To test the application:

  1. Make sure the backend server is running on port 5000
  2. Start the frontend with: cd dashboard && npm run dev
  3. Open the application in your browser
  4. Test the keyboard shortcuts on the NotesPage
  5. Test the timestamp input format (mm:ss or h:mm:ss)
